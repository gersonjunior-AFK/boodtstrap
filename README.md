# boodtstrap

Responsive frontend project focused on layout structure, component organization and interface experimentation using HTML, CSS and JavaScript.

## Overview

This repository contains a collection of frontend layout studies built to explore responsive design patterns and reusable interface structures.

The project was developed with simplicity in mind, keeping the code lightweight and easy to modify.

## Stack

* HTML5
* CSS3
* JavaScript

## Structure

```bash
.
├── css
├── js
├── assets
└── index.html
```

## Running Locally

Clone the repository:

```bash
git clone https://github.com/gersonjunior-AFK/boodtstrap.git
```

Open `index.html` in your browser.

## Goals

* Practice responsive layouts
* Improve frontend structure
* Experiment with interface styling
* Build reusable UI concepts
